Set a minimum and maximum order value for WooCommerce and displays custom messages on the cart and checkout pages.

== Description ==

**This plugin allows you to set a minimum and maximum order value in WooCommerce, displaying custom messages on the cart and checkout pages.** This helps control orders and optimize the user experience, ensuring customers do not place orders that are too small or too large.

### Plugin Features
- **Set a Minimum Order Value**: Define a minimum amount that customers must meet to place an order.
- **Set a Maximum Order Value**: Limit the maximum order amount to prevent excessive sales.
- **Custom Error Messages**: If the order does not meet the established conditions, error messages will be displayed on the cart and checkout pages.
- **Easy Configuration**: The admin interface allows you to easily enable and disable the minimum and maximum value functionalities.

### How to Use the Plugin
1. **Install and activate the plugin**.
2. **Go to WooCommerce > Order Value Settings** to configure the minimum and maximum values.
3. **Customize the error messages to suit your needs**.

### Requirements
- **WooCommerce**: This plugin requires WooCommerce to function.
- **PHP Version**: You need at least PHP 7.0 for proper plugin execution.

== Installation ==

1. **Upload the plugin file to the `/wp-content/plugins/` directory** or install it directly from the WordPress plugin repository.
2. **Activate the plugin from the Plugins menu in WordPress**.
3. **Configure the options in WooCommerce > Min/Max Order Value Settings**.
